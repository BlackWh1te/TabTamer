TabTamer

AI-powered browser tab organizer for Chrome

⚠️ Chrome Required

TabTamer is a Chrome extension. Please open this page in Google Chrome to install.

Installation Guide

1

Download TabTamer

Download the latest release from GitHub and extract the zip file.

Download TabTamer.zip
2

Open Chrome Extensions

Type this URL in a new tab or click below:

chrome://extensions/
3

Enable Developer Mode

Toggle the switch in the top-right corner labeled "Developer mode".

4

Load the Extension

Click "Load unpacked" and select the extension folder from the downloaded TabTamer folder.

TabTamer-master/extension/
5

You're Done! 🎉

TabTamer is now installed. Press Ctrl+Shift+Y to open it and tame your tabs!

AI-Powered

Smart 3-layer clustering engine automatically groups your tabs by topic.

Privacy-First

All processing happens locally. No data sent to servers.

Save Sessions

Save, load, and share tab sessions for different projects.
